# A Simple Student Grading Management System 

This system manages student records calculating General Weighted Average


## Description

GWA Calcuation
Formula: (Subject Grade x Units) or (Units or Sum of Grades x Total Units / Total Units)

## V2 Features

*Records Name
*Records ID #
*Records Grade per Subject
*Records Units per Subject
*Calculates General Weighted Average
*ID Digit Min/Max
*Enter before proceeding feature


### Dependencies

*Makefile Extension
*GUI made with GTK3.0 / GTK4.0
*VSCode
*WinGW/GCC Compiler

### Executing program

Run Terminal:
(Windows)
1. cd C:/path/to/folder
2. gcc main.c studentfunc.c -o student_app.exe
3. make
4. student_app.exe

Run Terminal:
(Linux)
1. cd /path/to/folder
2. ls (check if files are complete and it exist.)
3. gcc main.c studentfunc.c -o student_app.exe
4. make
5. ./student_app
